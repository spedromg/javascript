JavaScript Fundamentos

JavaScript é uma linguagem de programação multiparadigma (várias formas de desenvolver uma solução), leve, dinâmica e interpretada. JavaScript é a linguagem de programação da Web, utilizada tando nos navegadores, como em servidores desenvolvidos especificamente para interpretá-la, como por exemplo o popular node.js (https://nodejs.org/en/).
